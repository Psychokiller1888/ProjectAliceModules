# Tasmota

##### > WGET method
```bash
wget http://modules.projectalice.ch/Tasmota \
-O ~/ProjectAlice/system/moduleInstallTickets/Tasmota.install
```

```bash
./alice add module Psychokiller1888 Tasmota
```
